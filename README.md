This project applies machine learning techniques to predict the risk of heart disease using clinical and demographic data. The goal of this project is to analyze key health indicators, develop predictive models, and evaluate their effectiveness in identifying patients at risk of heart disease.

The dataset consists of patient health records containing medically relevant attributes, including:

- Maximum heart rate achieved
